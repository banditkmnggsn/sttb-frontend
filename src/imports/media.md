# Role
Expert UI/UX Designer for Digital Media Platforms and Universities.

# Task
Redesign the "Media & Sumber Daya" (Media & Resources) page for STT Bandung. The goal is to transform a cluttered list of links into a highly engaging, Netflix/Spotify-style media hub that categorizes Videos, Articles, and Podcasts beautifully.

# Visual Style
- **Theme:** Modern, Immersive, and Content-First.
- **Color Palette:** Deep Navy Blue (#002147), Dark Slate/Charcoal for background sections (to make video thumbnails pop), and Bright White/Gold for text and interactive elements.
- **Typography:** Inter (or similar clean Sans-Serif) for all UI elements to maintain a modern, app-like feel. Use bold weights for video/article titles.
- **Aesthetic:** Dark mode sections for videos, light mode for articles, featuring horizontal scrolling (carousels) and large, high-quality thumbnails.

# Page Structure & Components

1. **Header & Navigation:**
   - Standard unified navigation bar.
   - A clean Search Bar placed prominently in the header specifically for finding media.

2. **Hero Section (Featured Media):**
   - A large, cinematic "Featured Content" area.
   - Use a large video thumbnail (e.g., for "City TransForMission").
   - Overlay the Title, a brief description, a "Topic" badge (e.g., Urban Missions), and a prominent Gold "Play/Read Now" button.

3. **Media Format Tabs (Sticky Filter Bar):**
   - Clean horizontal tabs to switch between media types: "Semua (All)", "Video", "Artikel", "Podcast", "Buletin".
   - *Note: Selecting a tab should ideally filter the content below without reloading the page.*

4. **Trending / Latest Videos (Horizontal Carousel):**
   - Background: Dark Navy/Slate.
   - Section Title: "Video Terbaru" (Latest Videos).
   - Video Cards: 16:9 aspect ratio thumbnails with a subtle "Play" icon overlay. Below the thumbnail, include the Video Title, Date, and Duration (if applicable).

5. **Resource Library (Grid View):**
   - Background: Off-White (to differentiate from the video section).
   - Section Title: "Artikel & Publikasi" (Articles & Publications).
   - Content: A clean 3-column grid for Articles and Bulletins. 
   - Card Style: A small square image on the left, with the Title, Author, Date, and "Topic Tags" on the right. 

6. **Topic Exploration (Pill/Badge Cloud):**
   - Instead of a massive list of text checkboxes, create a beautiful, visually distinct section called "Jelajahi Topik" (Explore Topics).
   - Display topics as clickable, rounded buttons (Pills): [Apologetika] [Spiritualitas] [Kepemimpinan] [Pemuridan] [Pendidikan Teologi].
   - When a user hovers, the pill should invert colors (e.g., from White to Navy).

# Constraints
- Avoid vertical lists of checkboxes for filtering. Use horizontal pills, tabs, and carousels.
- Differentiate between "Video" and "Text/Audio" visually. Videos should use 16:9 thumbnail cards, while articles should use text-heavy cards.
- Ensure the layout is highly responsive; carousels should become swipeable on mobile.
