# Role
Expert UI/UX Designer for University Portals and Digital Libraries.

# Task
Redesign the "Perpustakaan" (Library) page for a Theological College (STT Bandung). The page must serve as a comprehensive "Digital Hub" that beautifully integrates book discovery, digital resources, library statistics, and public membership registration.

# Visual Style
- **Theme:** Scholarly, Inviting, and High-Tech.
- **Color Palette:** Deep Navy Blue (#002147) for authority, Warm Off-White/Cream (#FAF9F6) for background to mimic book pages, and vibrant Gold for CTAs.
- **Typography:** Serif (Playfair Display) for main headers to give an academic feel, and Sans-Serif (Inter) for UI elements, forms, and data.
- **Aesthetic:** "Bento Grid" for resources, clean overlapping cards, and subtle glassmorphism for sticky elements.

# Page Structure & Components

1. **Hero Section (Search & Discovery):**
   - Background: High-quality image of a modern, sunlit library interior.
   - Large prominent text: "Eksplorasi Pengetahuan & Teologi" (Explore Knowledge & Theology).
   - Core Element: A large, elevated Search Bar in the center (placeholder for OPAC/Catalog search) with a dropdown for "Title, Author, or ISBN".
   - Below the search bar, three quick-action pill buttons: "Katalog Fisik", "Perpanjang Mandiri", "Pustaka Pintar".

2. **Library Statistics (Trust Indicators):**
   - A clean, horizontal statistics banner over a Navy background.
   - Icons with numbers: 47,254+ Judul (Titles), 51,932+ Eksemplar (Copies), 39,913+ Sirkulasi (Circulation).

3. **Digital Resources Grid (Bento Style):**
   - Section Title: "Akses Jurnal & E-Library"
   - Use a sleek grid layout to display external resource partners. 
   - Cards should feature logos/names like: EBSCO, ATLA, PERPUSNAS, GARUDA, SINTA, GLOBETHICS. 
   - Add hover effects (slight lift and shadow) to indicate they are clickable links.

4. **Membership & Administration Section (Side-by-Side Layout):**
   - Section Title: "Layanan Keanggotaan Terbuka" (Open Membership Services).
   - **Left Column (Info Card):** A beautifully styled card outlining the Membership Terms (Deposit Rp 250.000, Annual Fee Rp 100.000) using custom checkmark icons for bullet points.
   - **Right Column (Form UI):** A modern, multi-step or clean single-column registration form inside a card. Include styled input fields for Name, Email, and modern "Drag & Drop" zones for file uploads (KTP, Pas Foto, Bukti Transfer). 
   - Add a toggle switch at the top of the form to switch between "Daftar Anggota" (Register) and "Bebas Pustaka" (Clearance).

5. **Book Highlights (Carousel/Slider):**
   - Section Title: "Rekomendasi Buku Bulan Ini" (Book Highlights of the Month).
   - 3-4 vertical book cards showing the Book Cover, Title, a short truncated synopsis, and a "Cek Ketersediaan" (Check Availability) button.

# Constraints
- The Membership Form must look like a modern SaaS application form, not a raw HTML form.
- Ensure clear visual hierarchy: Book Search first, Digital Resources second, Membership third.
