# Role
Expert UI/UX Designer for Academic & Institutional Websites.

# Task
Design a modern "Events and Activities" (Kegiatan) page for a Theological College (STT Bandung). The design should replace a traditional calendar list with a dynamic, visually engaging user experience.

# Visual Style
- **Theme:** Professional, Academic, and Trustworthy.
- **Color Palette:** Deep Navy Blue (#002147), Gold/Mustard accents for CTAs, and plenty of White Space.
- **Typography:** Use a sophisticated Serif for headings (e.g., Playfair Display) and a clean Sans-Serif for body and filters (e.g., Inter or Plus Jakarta Sans).
- **Aesthetic:** Minimalist "Bento Grid" or modern "Card-based" layout with soft shadows and rounded corners (16px).

# Page Structure & Components
1. **Header & Navigation:** - Transparent sticky glassmorphism navbar. 
   - Links: About, Academic, Admisi, Kehidupan Kampus.
   - Distinct "Login" button with a primary color fill.

2. **Hero Section:**
   - Title: "Kegiatan & Agenda Kampus"
   - Subtitle: "Stay updated with the latest seminars, workshops, and spiritual activities at STT Bandung."
   - Visual: Subtle abstract geometric pattern background.

3. **Advanced Filtering System:**
   - A horizontal scrollable filter bar for Categories: (Seminars, Admisi, Chapel, Academic Calendar, Research).
   - A minimalist Date Picker or "Upcoming This Month" toggle.

4. **Event Grid (The Core):**
   - Use a **Bento Grid** layout.
   - **Featured Event Card:** Double-width card for major events (e.g., "Dies Natalis" or "Open House") with a large background image, a "Featured" badge, and a "Register" button.
   - **Standard Event Cards:** Include a date badge (e.g., "14 MAR"), event title, time (e.g., 09:00 - 12:00), and a short "Category" tag.
   - Use high-quality placeholders for campus photography.

5. **Sidebar/Quick Info (Optional Desktop):**
   - A "Monthly Highlights" widget.
   - "Download Academic Calendar" CTA button.

6. **Footer:**
   - Minimalist footer with social media links and contact info.

# Constraints
- Ensure the layout is mobile-responsive.
- Avoid cluttered tables; use cards and vertical lists with clear hierarchy.
- Prioritize readability of event dates and titles.
